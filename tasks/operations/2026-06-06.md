# 2026-06-06 ウエイトトレーニング種目追加

- 作成日: 2026-06-06
- 担当: 山藤
- 期限: 2026-06-06

## 目的

ウエイトトレーニングメニュー内の種目に、以下の項目を追加して記録種目の精度を高める。
- ベンチプレス（ローテーションあり）
- ベンチプレス（ローテーションなし）
- ストレートバー

将来的に「種目追加」の運用タスクが発生した際に、対象をウエイトトレーニング、野球指標、体重・体組成のいずれかに分類できるようにする。

## 手順
1. `index.html` をテキストエディタで開く。
2. `ウエイトトレーニング記録` セクション内の `<select id="train-type">` を探す。
3. 現在ある種目オプションの一覧のうち、`<option value="その他">その他</option>` の直前に以下3行を追加する。
   - `<option>ベンチプレス（ローテーションあり）</option>`
   - `<option>ベンチプレス（ローテーションなし）</option>`
   - `<option>ストレートバー</option>`
4. `app.js` を開き、編集画面の `select` 要素 `id="edit-train-type"` を探す。
5. `edit-train-type` の `<select>` 内にも、上記と同じ3つの `<option>` 要素を追加する。
6. `app.js` の `train-type` を扱う箇所は通常そのままで問題ないが、`id="train-type"` と `id="edit-train-type"` の両方に新種目が含まれていることを確認する。
7. 保存後、ブラウザでアプリを開き、以下を確認する。
   - `ウエイトトレーニング記録` の種目選択リストに新しい3種目が表示される
   - 編集画面で既存トレーニングレコードを開いたときに、新種目が選択肢として表示される
8. もし表示されない場合は、`index.html` の `<option>` が正しい場所に挿入されているか、同じ文字列で重複がないか再確認する。
9. 問題がなければ、追加した種目の詳細や用途を運用ドキュメントに追記する。
10. 最後に、変更内容を保存して、テスト環境または本番環境で動作を確認する。
