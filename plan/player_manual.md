## はじめに

このシステムは、野球部の選手管理を簡単に行える Web アプリです。

- **選手は自分のアカウントを作成し、ログインして情報を確認できます**。
- **マスター（部長・コーチなど）は他の選手のロール（役割）を変更できる権限があります**。
- アプリは Firebase の Authentication と Firestore を利用しており、データはクラウドに安全に保存されます。

---

## アプリの起動方法

1. ブラウザを開き、以下の URL にアクセスします。`https://baseball-training-app-8982a.web.app`
2. 画面右上に「ログイン」ボタンが表示されます。まずは **新規登録** を行いましょう。

---

## 新規登録（選手登録）

1. **「ログイン」ボタン** をクリックし、モーダルが開いたら **「新しく選手を登録する」** を選択します。
2. 必要項目を入力します。
   - 名前（必須）
   - ポジション（例: 投手、捕手…）
   - 背番号（任意）
   - パスワード（8 文字以上）
3. 入力が完了したら **「登録」** ボタンをクリック。
4. 登録が成功すると、Firebase が自動で **synthetic email**（例 `player_xxxxx@baseball-tracker.internal`）を作成し、**UID** がプレイヤーのドキュメント ID として `players` コレクションに保存されます。
5. 登録直後は自動で **ログイン** され、メイン画面が表示されます。

---

## ログイン手順

1. **「ログイン」ボタン** → **「登録済みユーザーを選択」** ドロップダウンが表示されます。
2. 登録したときの **名前** が一覧に出ているので選び、パスワードを入力します。
3. **「ログイン」** をクリックすると、FireStore から自分の情報が取得され、サイドバーに **自分の名前・ポジション・背番号** が表示されます。
4. ログイン状態は `localStorage` に保存されているため、ブラウザを閉じても次回開いたときに自動復元されます（ただし別端末やプライベートモードでは再度ログインが必要です）。

---

## マスター権限（管理者）について

- **マスター** は `players` コレクション内の `role` フィールドが `"master"` に設定されたユーザーです。
- マスターは **サイドバー右上に「ロール変更」ドロップダウン** が表示され、他の選手のロールを `player` ⇄ `master` に切り替えることができます。
- **マスターになる手順**（部長・コーチが最初に実施）
  1. 任意の選手を通常通り登録する。
  2. Firebase コンソールの **Firestore → players → 該当 UID** を開く。
  3. `role` フィールドを追加し、値を `"master"` に設定して保存。
  4. アプリに再度ログインすれば、マスター用 UI が自動で有効化されます。

---

## 自分の情報の確認・変更

- 現在ログイン中の選手情報はサイドバーに常に表示されます。
- 背番号やポジションは **「選手情報編集」ボタン**（左下）から変更可能です（変更は即座に Firestore に反映されます）。
- パスワードを忘れた場合は、**Firebase のパスワードリセット** 機能は実装していませんので、コーチに新しいパスワードを発行してもらってください。

---

## 他の選手の情報閲覧 (マスターのみ)

- マスターがログインすると、サイドバーに **全選手一覧** が表示されます。
- 各選手のカードには **ロールバッジ**（`管理者` or `選手`）が付き、**ロール変更ドロップダウン** が表示されます。
- ドロップダウンで `管理者` を選ぶと、その選手の `role` が `master` に更新され、即座に権限が拡張されます。

## トラブルシューティング

| 症状                                       | 原因例                                                          | 対処方法                                                                                              |
| ------------------------------------------ | --------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| ログインできない                           | パスワードが間違っている                                        | コーチにパスワード再発行を依頼                                                                        |
| 登録したはずなのに画面に選手が表示されない | Firestore のルールがまだブートストラップ状態のまま              | `firebase deploy --only "firestore:rules"` が成功しているか確認。再デプロイ後にブラウザをリロード。 |
| マスター UI が表示されない                 | `players` の該当 UID に `role: "master"` が設定されていない | コンソールで手動で `role` フィールドを追加。                                                        |
| データが保存されない                       | ネットワーク環境がオフライン                                    | インターネット接続を確認し、再度操作。                                                                |
